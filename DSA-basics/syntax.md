# C++ Syntax

## Output
```cpp
cout << "Hello DSA";
cout << a;
cout << "Hello " << a << endl;
```

## Variables
```cpp
int a;
int a = 10;
```

## Input
```cpp
cin >> a;
cin >> a >> b;
```

## Operators
```
+  -  *  /  %    → arithmetic
=                 → assignment
==                → comparison
&&  ||            → logical AND / OR 
```

## If-else
```cpp
// 0 = false, anything else = true
if (a > 10) {
    cout << "hello";
} else if (a > 5) {
} else {}

// multiple conditions
if (a % 3 == 0 || a % 2 == 0) {}
```

## Loops
```cpp
for (int i = 0; i < 5; i++) {
    cout << "hello" << endl;
}
```
